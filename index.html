Proyectos del Curso

A continuación, se presentan los proyectos realizados como parte del curso EST334 - Estadística Espacial.

Primera Unidad - Fundamentos y Análisis Espacial

Posicionamiento de Información

Análisis de datos geoespaciales para determinar patrones de posicionamiento utilizando herramientas GIS con visualización de mapas interactivos generados en QGIS.

Ver

Muestreo Bootstrap

Aplicación de técnicas de muestreo bootstrap para estimar parámetros en datos espaciales, explorando variabilidad y confianza en resultados mediante R Spatial.

Ver

Gaussian Random Fields

Modelado de campos aleatorios gaussianos para analizar autocorrelación espacial mediante GeoPandas y Python, simulando procesos espaciales estocásticos.

Ver

Matrices de Pesos, Moran, Geary y Hotspots

Aplicación de cuatro técnicas de estadística espacial: Matrices de Pesos, Índice I de Moran, Índice C de Geary y Análisis de Hotspots con datos de gastos agrícolas de 90 distritos de Puno (ENA 2024).

Ver

Aplicación de Autocorrelación Espacial

Resumen de tres artículos científicos 2025 sobre heterogeneidad espacial en contextos ambientales y salud pública, convergiendo en el uso de autocorrelación espacial para identificar hotspots y coldspots.

Ver

Research Statement

Presentación del proyecto de investigación e intenciones para artículo científico basado en datos ENA Perú, con modelado de procesos de Cox espacio-temporal.

Ver

Repositorio GitHub

Evidencia de la creación de la página GitHub y el pull request del proyecto, documentando el control de versiones y la colaboración en el desarrollo del curso.

Artículo Científico - Procesos de Cox

Modelado de patrones agrícolas en Perú usando procesos log-Gaussianos de Cox (LGCP) con datos ENA 2022-2024. Análisis espacio-temporal de 174,040 eventos con clustering significativo y coeficientes validados (AIC = -810,737).
