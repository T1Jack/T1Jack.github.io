Proyectos del Curso

A continuación, se presentan los proyectos realizados como parte del curso EST334 - Estadística Espacial.

Posicionamiento de Información

Análisis de datos geoespaciales para determinar patrones de posicionamiento utilizando herramientas GIS. Este proyecto incluye la visualización de datos y mapas interactivos generados con QGIS.

Ver Proyecto

Muestreo Bootstrap

Aplicación de técnicas de muestreo bootstrap para estimar parámetros en datos espaciales. Implementado en R Spatial, este proyecto explora la variabilidad y confianza en los resultados.

Ver Proyecto

Gaussian Random Fields

Modelado de campos aleatorios gaussianos para analizar la autocorrelación espacial. Utiliza GeoPandas y Python para simular y visualizar procesos espaciales estocásticos.

Ver Proyecto
